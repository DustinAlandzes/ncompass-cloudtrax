#NCompass-Cloudtrax

#Description

#Install
```
  clone repository
  cd repository
  virtualenv env
  source env/bin/activate
  pip install -r requirements.txt
  python app.py
```
#Usage

```
  /
```

Shows table of ProbeRequests

```
  /receive
```

Receives HTTP Post request from Cloudtrax

```
  /test
```

Adds sample ProbeRequests to the database
